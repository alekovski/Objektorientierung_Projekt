# Object-Oriented_Project

This project contains a desktop e-book reader application developed using JavaFX that demonstrates Object-Oriented Programming (OOP) principles.

## Project Features

- Library Management: View, save, and load a collection of books.
- Book Details: Displays book information including title, author, and reading progress.
- Search Functionality: Filter books by title.
- File Handling: Uses object serialization to persist the library data to a library.dat file.
- Recently Viewed: Tracks and displays the last 10 books opened.
- OOP Principles: The project is structured with classes like Book and Library, showcasing encapsulation and state management.
